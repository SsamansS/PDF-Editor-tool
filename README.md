# PDF Editor Tool

Веб-инструмент для редактирования PDF-файлов с браузерным интерфейсом на основе PDF.js.

## Возможности

- Редактирование текста внутри PDF (замена, поиск)
- Объединение, разделение, извлечение страниц
- Удаление и поворот страниц
- Добавление водяных знаков
- Шифрование и расшифровка PDF
- Извлечение текста в .txt
- История изменений с поддержкой отмены (undo)

## Требования

- Python 3.10+

## Установка и запуск

### 1. Распаковка архива

Распакуй архив в удобную папку, например:

```
C:\projects\pdf-editor\
```

### 2. Создание виртуального окружения

```bash
cd C:\projects\pdf-editor
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск

```bash
python pdf_editor.py gui
```

Автоматически откроется браузер на `http://localhost:8765`

**Запуск без открытия браузера:**

```bash
python pdf_editor.py web --host 127.0.0.1 --port 8765 --no-browser
```

**Справка по CLI-командам:**

```bash
python pdf_editor.py -h
```

## Структура проекта

| Файл | Назначение |
|------|-----------|
| `pdf_editor.py` | CLI + запуск веб-сервера |
| `pdf_editor_web.py` | HTTP-сервер (порт 8765) |
| `pdf_core.py` | Ядро — все операции с PDF |
| `requirements.txt` | Зависимости Python |
| `static/` | Фронтенд (HTML, CSS, JS, PDF.js) |
